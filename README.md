# Генератор откликов

AI-инструмент для автоматической адаптации резюме и написания сопроводительных писем под конкретную вакансию. Один HTML-файл без зависимостей — открывается в браузере и работает.

## Как работает

1. Вставляешь текст вакансии
2. Нажимаешь **Сгенерировать**
3. Claude адаптирует резюме под требования вакансии и пишет сопроводительное письмо
4. Каждый результат автоматически проверяется на галлюцинации с оценкой в процентах

Промпты хранятся в Notion и подтягиваются при каждой генерации — обновляешь промпт в Notion, инструмент подхватывает без правки кода.

## Стек

- **Vanilla HTML/CSS/JS** — один файл, без фреймворков и сборщиков
- **Material Design 3** — токены, elevation через surface tint, Extended FAB, shimmer-скелетон
- **Claude API** (claude-sonnet-4) — генерация резюме и письма параллельными запросами
- **Notion API** — хранение системных промптов, подтягиваются при каждом запросе

## Настройка

1. Открой `job-generator.html` в любом редакторе
2. Найди константы в начале скрипта и вставь свои ключи:

```js
const CLAUDE_KEY = 'sk-ant-...';      // Claude API key
const NOTION_KEY = 'secret_...';      // Notion Integration token
const NOTION_RESUME_ID = '...';       // ID страницы с промптом резюме
const NOTION_LETTER_ID = '...';       // ID страницы с промптом письма
```

3. Открой файл в браузере или задеплой на сервер

## Деплой на VPS

```bash
scp job-generator.html user@server:/var/www/html/job-generator.html
```

## Формат промптов в Notion

Каждая страница содержит системный промпт для Claude:
- профиль кандидата с опытом и метриками
- алгоритм адаптации
- структуру вывода (резюме / письмо + блок проверки)
- правила и запрет галлюцинаций

Блок проверки в ответе Claude парсится отдельно и отображается под результатом — в текст резюме и письма не попадает.

## Возможности

- Параллельная генерация резюме и письма за один запрос
- Автоматическая детекция и парсинг блока проверки (PASSED / FAILED + оценка %)
- Shimmer-скелетон во время загрузки
- Светлая и тёмная тема с сохранением в localStorage
- Cmd+Enter для быстрой генерации
- Кнопка копирования — копирует только чистый текст без блока проверки

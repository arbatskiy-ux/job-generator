# Генератор откликов

AI-инструмент для автоматической адаптации резюме и написания сопроводительных писем под конкретную вакансию. Один HTML-файл без зависимостей — открывается в браузере и работает.

## Как работает

1. Вставляешь текст вакансии
2. Нажимаешь **Сгенерировать**
3. AI адаптирует резюме под требования вакансии и пишет сопроводительное письмо
4. Каждый результат автоматически проверяется на галлюцинации с оценкой в процентах

Промпты встроены в код — отредактируй их прямо в HTML если нужно кастомизировать логику или формат вывода.

## Стек

- **Vanilla HTML/CSS/JS** — один файл, без фреймворков и сборщиков (2042 строк)
- **Material Design 3** — 32 CSS-переменных, elevation через surface tint, shimmer-скелетон при загрузке
- **Несколько AI провайдеров:**
  - **Groq** — Llama 3.3 70B (мощный) или 3.1 8B (быстрый)
  - **OpenRouter** — Llama 3.3, DeepSeek V3, Qwen 2.5 (все бесплатные)
  - **Cloudflare AI** — Llama 3.1 8B или 3 8B (встроенный в Workers)
- **localStorage** — сохранение ключей, темы, резюме, истории

## Настройка

1. Открой \`job-generator.html\` в браузере
2. Выбери провайдера AI в настройках:
   - **Groq** — бесплатный, быстрый (Llama 3.3 70B или 3.1 8B)
   - **OpenRouter** — 3 бесплатные модели (Llama 3.3, DeepSeek V3, Qwen 2.5)
   - **Cloudflare AI** — требует Account ID и API ключ (Llama 3.1 или 3)
3. Введи API ключ для выбранного провайдера
4. Загрузи или вставь резюме в модальном окне (или выбери из сохранённых)
5. Добавь описание вакансии и нажми **Сгенерировать**

## Возможности

- **Параллельная генерация** резюме и письма за один API запрос
- **Автоматический парсинг** блока проверки галлюцинаций (PASSED ✅ / FAILED ❌ + оценка %)
- **Shimmer-скелетон** во время загрузки (плавная анимация загрузки)
- **Светлая и тёмная тема** с сохранением в localStorage
- **Горячие клавиши:**
  - **Cmd+Enter** (Mac) или **Ctrl+Enter** (Windows) — быстро сгенерировать
- **Копирование чистого текста** без служебных блоков
- **Банк резюме** — сохраняй варианты в localStorage без загрузки файлов
- **CSV экспорт** — скачивай историю в формате для Google Sheets/Excel
- **История генераций** — отслеживай все созданные резюме и письма
- **Адаптивный дизайн** — работает на мобильных и десктопе

## Банк резюме (фича v2)

Сохраняй и управляй несколькими вариантами резюме без загрузки файлов:

1. Введи или вставь текст резюме в модальное окно
2. Кнопка сохранения появится автоматически, когда текст превысит 10 символов
3. Нажми **💾 Сохранить как резюме** чтобы сохранить вариант
4. Сохранённые резюме появляются в выпадающем списке для быстрого выбора
5. Все варианты хранятся в localStorage браузера — сохраняются между сеансами

**Используй для:** Храни несколько версий резюме (универсальное, фокус на IT, управленческое) и быстро переключайся между ними.

## CSV экспорт (фича v2)

Экспортируй историю генераций в формат CSV для сохранения и анализа:

1. Нажми **⬆️ В Google Sheets** в панели истории
2. Скачивается CSV-файл со столбцами: Дата, Время, Компания, Сфера, Модель AI, Резюме, Письмо
3. Открой файл в Google Sheets, Excel или любом табличном редакторе
4. Все спецсимволы (кавычки, запятые) правильно экранированы

**Используй для:** Отслеживай отправленные приложения, анализируй какие промпты лучше работают, создавай отчёты о прогрессе в поиске работы.
